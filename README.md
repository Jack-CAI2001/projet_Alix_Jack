Projet angular avec le service api "https://www.themealdb.com/" fait en 2023.

C'est un site de recette de cuisine avec les fonctionnalités suivantes: 

-Barre de recherche

-Filtre par ingrédient

-Authentification

-Ajout en favoris

-Panier

avec le framework:
Angular v14 et
Angular Material v13
